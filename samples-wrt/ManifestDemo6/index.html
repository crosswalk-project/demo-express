Crosswalk Manifest Demo 6

This sample demonstrates mainfest members "icons" and "xwalk_launch_screen"

This demo shows App features can be configured by manifest.json

Download app named like "manifestdemo6.apk" from https://github.com/crosswalk-project/demo-express/releases (It might be included in .zip files in this page)

Check below features after install the app

  • Icon: A png format icon (blue background) with "Manifest PNG" text

Check below features after launch the app

  • Splash Screen: "Splash Screen Auto" text with blue background
  • Splash Screen: Ready when the window.screen.show() method was called
  • Splash Screen: Can switch screen between portrait and landscape modes
  • Xwalk_launch_screen members
    • Ready_when: The application readiness state at which to hide the launch screen
      • Custom:
        • The first visually non-empty paint has occurred
        • The window.screen.show() method was called
