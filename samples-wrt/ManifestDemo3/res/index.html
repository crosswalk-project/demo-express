Crosswalk Manifest Demo 3

This demo shows following features can be configured by manifest.json:
  • Icon: A bmp format icon (turquoise background color)
  • Xwalk_version: 1.0.0.1
  • Orientation: Landscape mode
  • Splash Screen: Turquoise background color
  • Splash Screen: Ready when the window.screen.show() method was called
  • Display: Opens the web application using the platform-specific convention for opening hyperlinks in the user agent (e.g., in a browser tab or a new window)
