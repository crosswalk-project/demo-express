Crosswalk Manifest Demo 3

This sample demonstrates mainfest members "icons", "orientation", "display" and "xwalk_launch_screen"

This demo shows App features can be configured by manifest.json

Download app named like "manifestdemo3.apk" from https://github.com/crosswalk-project/demo-express/releases (It might be included in .zip files in this page)

Check below features after install the app

  • Icon: A bmp format icon (turquoise background) with "Manifest BMP" text

Check below features after launch the app

  • Orientation: Landscape mode
  • Splash Screen: "Splash Screen Landscape" text with turquoise background
  • Splash Screen: The screen always stay the loading page
  • Display: Opens the web application using the platform-specific convention for opening hyperlinks in the user agent (e.g., in a browser tab or a new window)
  • Broswer: Opens the web application using the platform-specific convention for opening hyperlinks in the user agent (e.g., in a browser tab or a new window)
  • Xwalk_launch_screen members
    • Ready_when: The application readiness state at which to hide the launch screen
      • Custom:
        • The first visually non-empty paint has occurred
