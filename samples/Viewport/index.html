This sample demonstrates viewport with functionality of setting customized height and width, device height and width, scale, enabling or disabling user scalable to allow or disallow zoom in/out screen, sliding the screen by figure.

Test Step:
  1. Download app named like "ViewportTest.apk" from https://github.com/crosswalk-project/demo-express/releases (It might be included in .zip files in this page), then install it.
  2. Launch app "ViewportTest"
  3. Click the button in "ViewportTest" to test Viewport

This sample is designed by width, height, initial-scale, minimum-scale, maximum-scale and user-scalable property.
